# Rural Utility Cost: Dairy Economics Hub

A professional suite of technical decision-support tools for dairy production, feed efficiency, economics, and scenario planning. This module operates as a core component of the broader RuralUtilityCost.com ecosystem.

## 🎯 Purpose
The Dairy Hub provides dairy farmers, herd managers, agricultural lenders, and consultants with specialized tools to estimate milk production outputs, calculate Income Over Feed Cost (IOFC), assess break-even milk pricing, and plan for operations expansions.

## 🛠 Features & Tooling
- **Milk Production Forecasting:** Project yields based on current herd size and standard lactation seasonality curves.
- **Milk Pricing Calculator:** Estimate net payouts based on base prices coupled with component premiums (Butterfat, Protein).
- **Feed & IOFC:** Calculate daily feed efficiencies and derive margins-per-cow over fluctuating feed costs.
- **Break-Even Configurator:** Establish the required $ / CWT baseline covering operational, fixed, and feed costs.
- **Margin Coverage & Risk:** Model sensitivity and approximate exposure utilizing common USDA DMC-style variables.
- **Expansion Planner:** Assess capital expenditure (CapEx) against newly projected yield capacity to measure Return on Investment (ROI) and Simple Payback Period.
- **Favorites:** Readily pin operational parameters to local storage for quick access across sessions.

## 🖥 Tech Stack
- **Framework:** React 19 + TypeScript
- **Build Tool:** Vite 6
- **Layout & Syling:** Tailwind CSS V4
- **Routing:** React Router v6
- **Data Visualization:** Recharts
- **Icons:** Lucide React

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- `npm` or `yarn` package manager

### Local Development
1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start Development Server:**
   ```bash
   npm run dev
   ```
   *The server will become available on port `3000` or the first available port context.*

3. **Build for Production:**
   ```bash
   npm run build
   ```
